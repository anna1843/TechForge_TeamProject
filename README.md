
# SpringBoot-Project-SoleManager
스프링 부트 -> 프리랜서 중개 에이전시 그룹웨어
<br>

## 🖥️ 프로젝트 소개
1. 프리랜서와 회사 간의 적합한 프로젝트를 찾아 매칭하는 서비스 제공
2. 계약서 작성, 결제 처리 및 금융 관련 서비스를 제공
3. 프리랜서와 회사 간의 원활한 커뮤니케이션을 제공
다음과 같은 역할을 하는 그룹웨어에 서비스를 추가한 프로젝트입니다.
<br>
